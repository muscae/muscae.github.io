How to use Ivona TTS (Text To Speech) on Android

  1. Download Ivona Kendra application from apkpure, this is a xapk application which has the application and the voice data inside it. Place the file on the root of the Internal Memory or SD Card.

  2. Usually the file is downloaded in the Downloads/ folder, move it to root of Internal Memory, so that xapk installer can find it.

  3. Download and install xapk application from xapk pure application, Click OK

  4. If you get this prompt - click SETTINGS

  5. Select Unknown sources

  6. You will get this prompt, click OK

  7. Unknown sources will be enabled now.

  8. Go back to Downloads/ and open the xapk pure application. Now you will get the Install prompt. Click INSTALL

  9. ..after installation, click OPEN

  10. Give permissions to the application

  11. Horizontally scroll, the menu and click APK/XAPK MANAGEMENT

  12. Install the IVONA Kendra US English package. By clicking INSTALL

  13. Click INSTALL

  14. After installation of application open the application and accept the licence.

  15. Install the Install IVONA Text-to-Speech HQ

  16. Open with Google Play Store, and install. If your device is not compatible with the application - download the apk from here.

  17. Install

  18. You should be able to see these three applications, You can remove the APKPure application.

  19. Go to Settings > SYSTEM & DEVICE > Additional settings > Language & input > Text-to-speech output, and select IVONA

  20. Go back to the Ivona Kendra application and you can then click Play

This completes the installation of the Ivona TTS, you can install any TTS application and direct it to use this voice.
